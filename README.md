# 🎙️ مینی اپ پادکست تلگرام

یک مینی اپ زیبا و حرفه‌ای برای نمایش و پخش قسمت‌های پادکست مستقیماً از کانال تلگرام.

---

## 📁 ساختار فایل‌ها

```
podcast-miniapp/
├── index.html       ← رابط کاربری اپ (دست نزن!)
├── podcasts.js      ← فایل مدیریت قسمت‌ها (اینجا ویرایش کن)
└── README.md        ← این فایل
```

---

## 🚀 راه‌اندازی اولیه (یک‌بار انجام می‌دی)

### مرحله ۱: آپلود در GitHub

1. به [github.com](https://github.com) برو و وارد حسابت بشو
2. روی **New repository** کلیک کن
3. نام مخزن را بنویس (مثلاً `my-podcast`)
4. **Public** انتخاب کن
5. هر دو فایل `index.html` و `podcasts.js` را آپلود کن

### مرحله ۲: فعال کردن GitHub Pages

1. در مخزن گیت‌هاب، برو به **Settings** → **Pages**
2. در بخش **Source** گزینه **Deploy from a branch** انتخاب کن
3. شاخه `main` و پوشه `/ (root)` را انتخاب کن
4. **Save** بزن
5. بعد از ۱-۲ دقیقه، لینک اپت می‌آد:  
   `https://username.github.io/my-podcast`

### مرحله ۳: ساخت بات تلگرام

1. در تلگرام سرچ کن: `@BotFather`
2. بنویس `/newbot` و نام بات بده
3. **توکن** دریافتی را کپی کن (مثلاً `1234567890:ABCdef...`)

### مرحله ۴: ثبت مینی اپ

1. در BotFather بنویس `/newapp`
2. بات خودت را انتخاب کن
3. نام اپ، توضیح، عکس و... بده
4. وقتی **Web App URL** خواست، لینک GitHub Pages را وارد کن

### مرحله ۵: اضافه کردن دکمه به بات (اختیاری)

در BotFather:
```
/setmenubutton
```
یا می‌تونی از طریق پیام در کانال اینلاین باتو را صدا کنی.

## ➕ اضافه کردن قسمت جدید (خیلی ساده!)

### مرحله ۱: گرفتن لینک فایل از تلگرام

1. وارد کانالت بشو
2. پیامی که فایل صوتی داره پیدا کن
3. روی پیام کلیک راست کن (یا در موبایل نگه‌دار)
4. **Copy Link** یا **Copy Message Link** بزن
5. لینک یه چیزی مثل این می‌شه:  
   `https://t.me/your_channel/7`

### مرحله ۲: ویرایش podcasts.js

فایل `podcasts.js` را باز کن و داخل آرایه `PODCASTS` یک آبجکت جدید اضافه کن:

```javascript
{
  id: 6,                                          // شماره یکتا (از آخرین id+1 کن)
  title: "عنوان قسمت جدید",
  description: "توضیح کوتاه و جذاب از محتوای قسمت",
  category: "tech",                               // tech | business | science | personal
  duration: "42:15",                              // مدت زمان
  date: "۱۴۰۳/۰۵/۰۱",                           // تاریخ انتشار
  episode: 6,                                     // شماره قسمت
  telegramLink: "https://t.me/your_channel/7",   // ← لینک تلگرام اینجا
  tags: ["تگ۱", "تگ۲"],                         // تگ‌های مرتبط
},
```

### مرحله ۳: آپلود در GitHub

**روش اول (ساده‌تر - وب):**
1. وارد مخزن گیت‌هاب بشو
2. روی `podcasts.js` کلیک کن
3. دکمه ✏️ (edit) بزن
4. تغییرات را اعمال کن
5. **Commit changes** بزن

**روش دوم (سریع‌تر - GitHub Desktop):**
1. GitHub Desktop دانلود کن
2. مخزن را clone کن
3. فایل را ویرایش کن
4. Commit و Push کن

## ⚙️ شخصی‌سازی

در بالای فایل `podcasts.js` اطلاعات کانالت را تغییر بده:

```javascript
const PODCAST_CONFIG = {
  channelName: "نام پادکست تو",
  channelUsername: "@your_channel",
  channelDescription: "توضیح درباره پادکست",
  hostName: "نام مجری",
};
```

### اضافه کردن دسته‌بندی جدید

در آرایه `CATEGORIES` اضافه کن:
```javascript
{ id: "history", label: "تاریخ", icon: "🏛️" },
```

## 🔧 رفع مشکل

**اپ صدا پخش نمی‌کند؟**
- لینک تلگرام را چک کن (باید پابلیک باشه)
- روی دکمه "تلگرام" بزن تا فایل مستقیم در تلگرام باز بشه
- مطمئن بشو کانال Public هستش

**تغییرات نشان داده نمی‌شه؟**
- چند دقیقه صبر کن (GitHub Pages کمی طول می‌کشه)
- Cache مرورگر را پاک کن
